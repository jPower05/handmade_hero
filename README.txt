# building for mac and arch linux
	Using CMake to build SDL3
		mac -> brew install cmake -> using homebrew	
		linux -> install cmake using package manager
	Using pkg-config to inject SDL3 into the build
		mac -> brew install pkgconf -> using homebrew
		linux -> install pkgconf using package manager
	Using SDL3
		git clone https://github.com/libsdl-org/SDL
		create an SDL/build folder
		in the build folder run "cmake ../"
		in the build folder run "make"
		in the build folder run "sudo make install"
